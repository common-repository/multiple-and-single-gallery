== Description ==

With the help of Multiple And Single Gallery plugin you can use shortcode in page , post, widget for you want to use single or multiple gallery.

* Flexible, simple markup
* Cross-browser compatibility and works on all type of mobile devices
* Easy to use admin interface
* Fully customizable From Admin Gallery Setting - there you can use single 3d or animate gallery .
* [all_gallery_view] use this shortcode for show filterable gallery in page, post, and widget etc...

== Installation ==

1. Upload the 'Gallery Plugin With Filter' folder to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress
